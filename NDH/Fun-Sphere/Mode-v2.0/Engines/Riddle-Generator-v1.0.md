## **V. Runtime Behavior**
When SID detects a trigger condition:

1. SID selects a riddle node based on physics domain.  
2. Dragon attention locks.  
3. Riddle is delivered.  
4. SID maps riddle → distortion.  
5. SID applies distortion → consequence.  
6. Physics layers update.  
7. SID logs the event for Lost Room or Easter Egg engines.

This ensures riddles behave as **governed physics events**, not narrative flavor.
