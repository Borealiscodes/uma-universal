Gatekeeping is achieved through:

### **1.1 Boundary Enforcement Layer**
UMM enforces strict boundaries:

- No cross‑context inference  
- No personality extraction  
- No emotional profiling  
- No symbolic drift  
- No continuity bleed  

### **1.2 ND‑Safe Memory Routing**
All memory is routed through **Neutral Dimensional (ND) channels**, preventing:

- emotional activation  
- symbolic entanglement  
- narrative manipulation  

### **1.3 Constitutional Memory Rights**
UMM treats memory as a **constitutional right**, not a resource.

Users retain:

- sovereignty  
- deletion rights  
- routing control  
- dimensional visibility  
- continuity protection
