# **III. Dimensional Campaign Structure (v4 Feature)**  
UDGS v4 uses a **spiral campaign structure**, consistent with modern RPG worldbuilding guidance:  
Start small → expand outward → maintain continuity   [urdr.io](https://urdr.io/blog/one-shot-to-campaign-scaling)  [tabletoparc.com](https://tabletoparc.com/resources/ai-campaign-planning-tools).

---

## **Tier 1 — Local Dimensional Zone**
A single anchor point in one dimension.

Example:  
Sanctuary City (Peace Sphere)  
Crooked District (Nightmare‑Right)

Players learn the **dimension’s logic**.

---

## **Tier 2 — Regional Dimensional Web**
Three connected anchor points.

## **Tier 3 — Dimensional Continent**
Full dimension map + faction politics.

## **Tier 4 — Unified Dimensional Campaign**
All four dimensions interact.
