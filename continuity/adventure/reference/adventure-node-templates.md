# 📄 **Adventure Node Templates**  
### Continuity Adventure System — Forest of Echoes Edition  
Reusable templates for creating World Nodes, Character Nodes, and Story Nodes.

---

## 🌲 1. Purpose  
This document provides **clean, ready‑to‑use templates** for contributors creating adventure nodes.

Each template is:

- ND‑paced  
- reversible  
- symbolic  
- gentle  
- continuity‑safe  

These templates ensure consistency across all adventures.

---

# 🌿 **World Node Template**  
A World Node defines the environment, its symbolic behaviors, and its pacing.

```
World Node: [Name]
- biome: [list of environmental elements]
- stability membrane: [how stable or shifting the environment feels]
- symbolic hazards: [non-harmful symbolic challenges]
- stillness nodes: [locations of absolute quiet or reset points]
- resonance behavior: [how sound, light, or echoes behave here]
- pacing: [slow, gentle, breathable]
```

### Example  
```
World Node: Mosslight Clearing
- biome: bioluminescent moss, resonance trees
- stability membrane: gently shifting fog
- symbolic hazards: soft déjà vu
- stillness nodes: center clearing
- resonance behavior: moss pulses with emotional tone
- pacing: slow, breathable
```

# 🌫️ **Character Node Template**  
A Character Node defines the entity exploring the world.

```
Character Node: [Name]
- intent: [what the character wants]
- tone: [emotional or sensory tone]
- ability: [symbolic ability]
- ecology link: [how the character interacts with the environment]
- reversible traits: [trait A ↔ trait B]
- safety header: ND-paced, reversible, non-coercive
- pacing: [gentle, slow, breathable]
```

### Example  
```
Character Node: The Listener
- intent: follow the softest sound
- tone: gentle, observant
- ability: hear future echoes
- ecology link: mosslight resonance
- reversible traits: hesitant ↔ adventurous
- safety header: ND-paced, reversible, non-coercive
- pacing: slow, breathable
```

# 🔮 **Story Node Template**  
A Story Node defines a reversible narrative moment.

```
Story Node: [Name]
- soft quest: [gentle objective]
- reversible scene: [how the scene resets or rewinds]
- pacing membrane: [how slow or breathable the moment is]
- gentle arc: [emotion or understanding shift]
- stillness checkpoint: [where the scene pauses]
- symbolic echo: [optional symbolic element]
```

### Example  
```
Story Node: Whisper in the Clearing
- soft quest: follow the faint hum
- reversible scene: hum fades if overwhelmed
- pacing membrane: slow, steady
- gentle arc: confusion → clarity
- stillness checkpoint: mosslight pulse
- symbolic echo: faint harmonic shimmer
```
