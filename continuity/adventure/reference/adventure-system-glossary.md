# 🌲 **A — Adventure Architecture**

### **Adventure Mode**  
A reversible, ND‑paced interaction style defining *how* exploration occurs.  
Examples: **World‑Builder**, **Realm‑Walker**.

### **Adventure Node**  
A structural unit of an adventure: World Node, Character Node, Story Node.

### **Adventure Loop**  
The reversible tri‑path engine:  
**World → Character → Story → World → …**

### **Adventure Generator**  
The system that transforms fragments into nodes, chapters, and symbolic overlays.
