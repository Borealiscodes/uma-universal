# 📘 **Adventure System — Membrane Template Set README**  
### Contributor Guide — How to Use the Pacing, Hazard, Stability, and Cosmology Membrane Templates

The **Membrane Template Set** provides four ready‑to‑use templates for designing membranes inside the Adventure System.  
Each membrane template is:

- ND‑safe  
- reversible  
- symbolically aligned  
- structurally consistent  
- contributor‑friendly  

This README explains how to use the templates, how they fit into the Adventure System, and how contributors should integrate them into new or existing components.

# 🌱 **1. Purpose of This Folder**

This folder contains the **Adventure System Membrane Template Set**, which provides scaffolding for designing new membranes or extending existing ones.

Membranes regulate:

- pacing  
- hazard softening  
- stability  
- cosmology alignment  

They ensure all motion remains gentle, reversible, and ND‑safe.

# 📂 **2. Folder Contents**

Each membrane type has its own template file:

- **Pacing Membrane Template**  
- **Hazard Membrane Template**  
- **Stability Membrane Template**  
- **Cosmology Membrane Template**  

These four files together form the **Membrane Template Set**.

# 🧩 **3. How to Use These Templates**

### **3.1 When creating a new membrane**
Choose the template that matches your membrane type:

- pacing → story / narrative  
- hazard → world / resonance  
- stability → character / identity  
- cosmology → symbolic / alignment  

Copy the template into your new membrane file and fill in:

- purpose  
- functions  
- components  
- geometry  
- ND‑safe constraints  
- integration points  
- validation checklist  

### **3.2 When extending an existing membrane**
Use the template as a reference to ensure:

- reversible motion  
- stillness anchoring  
- symbolic geometry coherence  
- ND‑safe constraints  
- integration consistency  

### **3.3 When auditing a membrane**
Use the template alongside:

- **Validation Checklist**  
- **Audit Protocol**  
- **Audit Summary Template**  

This ensures membrane behavior remains aligned with system cosmology.

# 🔁 **4. ND‑Safe Membrane Requirements**

All membranes must follow ND‑safe rules:

- **No Agency**  
- **No Emotional Causation**  
- **No Acceleration**  
- **Reversible Motion**  
- **Stillness Anchor**  

These constraints ensure membranes remain gentle, safe, and non‑overloading.

# 🌌 **5. Symbolic Geometry (Coherent, Not Eldritch)**

Membrane geometry is symbolic, not literal.  
It must match membrane purpose:

- **wave** → pacing  
- **spiral** → hazard stabilization  
- **point** → centering  
- **ray** → cosmology clarity  

Geometry is coherent when it aligns with:

- membrane purpose  
- resonance flow  
- tone field  
- reversible motion  
- stillness anchoring  

And yes — it stays coherent **without** summoning cosmic horrors.
