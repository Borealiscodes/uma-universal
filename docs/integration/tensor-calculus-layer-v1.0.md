# **2. Tensor Objects**

NDH uses four primary tensors:

### **2.1 Resonance–Continuity Tensor \(T_{ij}\)**  
Captures how resonance amplitude interacts with continuity flow.

```
T_ij = R_i * C_j
```

Where:

- \(R_i\) = resonance amplitude along axis \(i\)  
- \(C_j\) = continuity flow along axis \(j\)

ASCII:

```
Resonance → Continuity
      ↓         ↓
      T_ij = R_i C_j
```

---

### **2.2 Geometry–Curvature Tensor \(G_{ij}\)**  
Defines manifold curvature under triangulation + altitude geometry.

```
G_ij = ∂(TV_i) / ∂(ALT_j)
```

Where:

- \(TV_i\) = triangulation vector  
- \(ALT_j\) = altitude ring geometry  

ASCII:

```
Triangulation → Altitude
      ↓           ↓
      G_ij = dTV_i / dALT_j
```

---

### **2.3 Stability–Envelope Tensor \(S_{ij}\)**  
Defines how stability envelopes deform under resonance + Omega anchoring.

```
S_ij = Ω_i * R_j
```

Where:

- \(Ω_i\) = Omega anchor strength  
- \(R_j\) = resonance amplitude  

ASCII:

```
Omega → Resonance
      ↓       ↓
      S_ij = Ω_i R_j
```

---

### **2.4 Interlock–Coupling Tensor \(I_{ij}\)**  
Defines coupling between manifolds during multi‑runtime integration.

```
I_ij = ∂(Interlock_i) / ∂(Manifold_j)
```

ASCII:

```
Interlock → Manifold
      ↓         ↓
      I_ij = dI_i / dM_j
```
