# **2. Sweep Overview**

The stability sweep consists of **five phases**, executed jointly by:

- SIAP  
- Safeguards  
- Safety Net  

These phases validate the entire integration surface.

### **Five Sweep Phases**
1. **Structural Sweep** — adjacency integrity  
2. **Traversal Sweep** — traversal‑integration correctness  
3. **Isolation Sweep** — cross‑plane boundary enforcement  
4. **Drift Sweep** — drift‑vector detection  
5. **Quarantine Sweep** — anomaly containment  

All phases must pass for integration‑plane stability certification.

---

## **Phase 1 — Structural Sweep**  
**SIAP validates structural adjacency between SYS‑CHS and TRV‑CHS‑OL.**
