## **1. Purpose**

The Pin Registry provides a **global index** of all pins across UMA‑Universal.  
Pins remain **inside their respective artifacts**, and the Registry stores:

- metadata  
- location  
- type  
- timestamps  
- hash lineage  

This registry does **not** relocate pins.  
It only indexes them.
