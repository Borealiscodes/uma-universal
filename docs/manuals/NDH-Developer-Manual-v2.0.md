The NDH Developer Manual v2.0 defines the architectural foundations, manifold structures, attractor types, stability principles, and traversal behaviors that govern the NDH system. It is the authoritative reference for all NDH subsystems:

- Serenity‑Manifold  
- Simulation‑Stack
