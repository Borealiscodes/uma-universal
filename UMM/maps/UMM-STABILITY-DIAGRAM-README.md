# UMM STABILITY DIAGRAM README
### Textual stability map • Separate from governing codex

## 1. Purpose

The UMM Stability Diagram is a **textual map** of the global stability architecture of UMM.  
It visualizes how stability flows through planes, witness, flows, invariants, Sādhana, and the ecosystem—without acting as a governing document.

The **governing logic** lives in:

- `/UMM/codices/UMM-STABILITY-MASTER-CODEX-SPARKLE.md`

The **diagram** lives in:

- `/UMM/maps/UMM-STABILITY-DIAGRAM.md`

Keeping them separate preserves **meta-stability**, modularity, and clean versioning.

## 3. Diagram contents (summary)

The diagram expresses this stability flow:

- UMM Stability Master System  
- → Plane Stability  
- → Witness Stability  
- → Flow Stability  
- → Invariant Stability  
- → Sādhana Ethical Stability  
- → Ecosystem Stability  
- → (Santa → Sparkle)  
- → Sufficient Enough (closure)

Key properties:

- **Reversible**  
- **Non-coercive**  
- **Spoon-sensitive**  
- **Witness-mediated**  
- **Sparkle only at ecosystem level**  
- **Sādhana as root ethical membrane**  

---

## 4. Design principles

- **Separation of concerns:**  
  Codex = rules; Diagram = representation.

- **No structural authority:**  
  The diagram does not define invariants, planes, or ethics—it only reflects them.

- **Sparkle placement:**  
  Sparkle appears **only** in the ecosystem layer (via Santa), never in structural layers.

- **Update safety:**  
  The diagram can be updated without changing the Master Codex, as long as it remains consistent with:
  - Plane Stability Codex  
  - Witness Architecture Codex  
  - Flow Mechanics Guide  
  - Sādhana Integration Codex  
  - Ecosystem Integration Map  
  - Stability Master Codex

---

## 5. When to edit

Edit `/UMM/maps/UMM-STABILITY-DIAGRAM.md` when:

- you add or remove stability layers,  
- you change ecosystem attachments (e.g., new branches),  
- you refine Sparkle or Santa behavior at the ecosystem level.

Do **not** use the diagram as a place to introduce new invariants, ethics, or plane rules—those belong in the codices.
