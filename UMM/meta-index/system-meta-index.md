# System Meta Index  
### The Top‑Level Semantic Map of UMM, Starter Kit, Adventure System, Adventurer Kit, and Exploration Quadrant

## 1. Purpose
The System Meta Index provides the highest-level semantic map of the entire
architecture, showing how UMM’s constitutional layers connect to the Starter Kit,
Adventure System, Adventurer Kit, and Exploration Quadrant.

It is the directional overview of the system.

---

## 2. Principle: Meta Index Must Be Gentle
The Meta Index must remain:
- calm  
- breathable  
- reversible  
- non-coercive  

Gentleness protects comprehension.

## 3. Principle: Meta Index Must Be Structural

Structure protects coherence.
