## 🌸 **1. Soft Presence Template**  
### Category: Soft Presence  
### Plane: Quiet / Continuity / External

```
# Scenario: Soft Presence — Template
Trigger: <user invocation / module request>
Plane: <Quiet / Continuity / External>
Mode: Soft Presence Mode

## Invitation Phase
- Soft Drift enters.
- Micro‑Sparkle flickers once.
- Plush Pulse grounds the scene.
- Lantern Blink stabilizes.

## Presence Phase
- Cushioned Quiet establishes background.
- Petal Pause stabilizes pacing.
- Star Drift provides continuity.

## Adjustment Phase (optional)
- Petal Slow‑Fall eases tempo.
- Lantern Dim‑Pause softens tone.
- Mint Sway clears heaviness.

## Exit Phase
- Lantern Fade‑Out.
- Petal Dissolve.
- Bubble Lift‑Away.

Safety: symbolic, reversible, non‑directive.
```
