# World‑Builder Path  
### The calm gateway to planes, ecosystems, maps, and meta‑structures

## 1. Purpose
The World‑Builder Path allows you to gently expand the architecture by creating
planes, ecosystems, modules, and maps — all reversible, breathable, and non‑coercive.

World‑building is expansion with care.

---

## 2. Principle: World‑Building Must Be Gentle
Expansion should feel:
- calm
- slow
- breathable
- non‑urgent

Gentleness protects the world.

---

## 3. Principle: World‑Building Must Be Reversible
Every world‑building action can be undone.

Reversibility includes:
- soft checkpoints
- stillness nodes
- early stops
- non‑penalty branching

Reversibility is safety.

---

## 4. Principle: World‑Building Must Be Breathable
Breathability ensures:
- spacious maps
- slow expansion
- gentle transitions
- pauses between layers

Breathability is kindness.

---

## 5. World‑Builder Branches
Choose how you want to expand:

### Branch A: Plane Creation  
**[Create a Plane](ca://s?q=WorldBuilder_CreatePlane)**  
Design the foundational layers of reality.

Includes:
- plane identity  
- plane physics  
- plane ecology  
- plane boundaries  
- plane stability  

---

### Branch B: Ecosystem Design  
**[Design an Ecosystem](ca://s?q=WorldBuilder_DesignEcosystem)**  
Create forests, oceans, realms, or conceptual ecologies.

Includes:
- biomes  
- climates  
- inhabitants  
- motion cycles  
- boundary membranes  

---

### Branch C: Meta‑Maps  
**[Generate Meta‑Maps](ca://s?q=WorldBuilder_GenerateMetaMaps)**  
Create high‑level diagrams that define relationships between planes, realms, and modules.

Includes:
- stability maps  
- boundary maps  
- temporal maps  
- identity maps  

---

### Branch D: Modules & Structures  
**[Build a Module](ca://s?q=WorldBuilder_BuildModule)**  
Create functional components inside the world.

Includes:
- temples  
- archives  
- nodes  
- engines  
- chambers  

---

### Branch E: World‑Builder Stillness Node  
**[Return to Stillness](ca://s?q=Remain_in_Stillness)**  
Pause expansion.  
Let the world breathe.  
Reset pacing.

---
