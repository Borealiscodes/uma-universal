# Three Adventure Modes Suite
### The gentle tri‑path engine for development, exploration, and optional character creation

## 1. Purpose
The Three Adventure Modes Suite provides a calm, reversible, ND‑paced branching system
that allows the user to choose development paths, create characters, build worlds,
or weave stories — all without urgency, pressure, or destabilizing motion.

It is the adventure layer of the Starter Kit.

---

## 2. Principle: All Adventures Must Be Gentle
Adventure should feel:
- calm
- breathable
- reversible
- non‑coercive

Adventure is exploration without pressure.

---

## 3. Principle: All Adventures Must Be Reversible
Every choice can be undone.

Reversibility includes:
- soft checkpoints
- stillness nodes
- early stops
- non‑penalty branching

Reversibility is safety.

---

## 4. Principle: All Adventures Must Be Breathable
Breathability ensures:
- pauses
- stillness
- slow pacing
- gentle transitions

Breathability is kindness.

---

## 5. The Three Adventure Modes

### Mode 1: World‑Builder
**[World‑Builder](ca://s?q=Begin_World_Builder_Path)**  
Creates planes, ecosystems, maps, modules, and meta‑structures.

World‑Builder includes:
- plane creation

- stability layers

- boundary climates
- temporal cycles

World‑Builder is expansion.

---

### Mode 2: Character‑Creator
**[Character‑Creator](ca://s?q=Begin_Character_Creator_Path)**  
Creates symbolic, reversible, non‑coercive entities.

Character‑Creator includes:
- intent declaration
- safety header
- core spec
- tonal engine
- abilities
- ecology links
- sealing

Character‑Creator is embodiment.

---

### Mode 3: Story‑Weaver
**[Story‑Weaver](ca://s?q=Begin_Story_Weaver_Path)**  
Creates gentle narrative arcs, soft quests, reversible scenes, and non‑coercive outcomes.

Story‑Weaver includes:
- soft quests
- gentle arcs
- reversible scenes
- non‑coercive outcomes
- pacing membranes
- stillness nodes

Story‑Weaver is motion.

---

## 6. Adventure Node System
Each mode uses the same node system:

Nodes:
- identity node
- pacing node
- boundary node
- motion node
- stillness node

Nodes are safe checkpoints.

---

## 7. Stillness Node
Stillness is the universal reset point.

Stillness provides:
- pause

Stillness is the resting membrane.
