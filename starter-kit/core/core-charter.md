# Core Charter  
### The Foundational Identity and Purpose of Your Project

## 1. Purpose
The Core Charter defines the heart of your project.  
It explains why the project exists, what it hopes to become, and the gentle principles that guide its growth.

This document is calm, friendly, and non-coercive.  
It is a beginning, not a boundary.

---

## 2. Project Identity
Describe the identity of your project in a soft, simple way.

Examples:
- “This project is a space for exploration.”  
- “This project is a tool for clarity.”  
- “This project is a gentle architecture for ideas.”  

Identity is not a demand.  
It is a feeling.

---

## 3. Project Purpose
State the purpose of your project in one or two clear lines.

Examples:
- “The purpose of this project is to organize ideas with kindness.”  
- “The purpose of this project is to build a stable foundation for future work.”  
- “The purpose of this project is to create a safe, structured place for creativity.”  

Purpose should feel light, not heavy.

---

## 4. Project Values
List the values that guide your project.

Examples:
- Clarity  
- Gentleness  
- Stability  
- Curiosity  
- Non-coercion  
- Respect for pacing  

Values are the colors on your palette.

---

## 5. Project Tone
Describe the tone you want your project to have.

Examples:
- calm  
- friendly  
- steady  
- welcoming  
- patient  

Tone is the mood of the canvas.

---

## 6. Project Boundaries
Define soft boundaries that keep your project safe and stable.

Examples:
- “This project avoids urgency.”  
- “This project respects pacing.”  
- “This project grows in small, happy strokes.”  
- “This project does not collapse into complexity.”  

Boundaries are gentle edges, not walls.

---

## 7. Project Commitments
List commitments that feel supportive, not demanding.

Examples:
- “We commit to small steps.”  
- “We commit to clarity.”  
- “We commit to kindness in structure.”  
- “We commit to stopping early.”  

Commitments should feel like friendly promises.

---

## 8. Summary
The Core Charter is the heart of your project.  
It defines identity, purpose, tone, values, and boundaries in a gentle, Bob-Ross-style way.
