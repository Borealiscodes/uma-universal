# **3. How to Interpret Drift Rates**

Drift rate:

\[
D(t)=\frac{\partial M}{\partial t}
\]

Interpretation:

### **D(t)=0**  
Meaning stable.  
Operator posture: No action.

### **D(t)<0**  
Meaning decaying.  
Operator posture: Amplify meaning harmonic.

### **D(t)\ll 0**  
Meaning collapsing.  
Operator posture: Activate Citadel + Reactor.

Drift rate is the **earliest warning signal**.
