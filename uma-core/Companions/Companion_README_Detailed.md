**Path:** `uma-core/Companions/Decompression_Companion.md`

### Modes  
- **Quiet Mode** — minimal language, low load  
- **Hybrid Mode** — gentle grounding  
- **Narrative Mode** — soft story‑based decompression  

### Core Functions

- Supports transitions between IC/OC
