# 🌱 **1. Instructor Principles**
UMM instructors follow five constitutional principles:

- **Reversible** — every activity can be undone  
- **Symbolic** — nothing is literal or emotional  
- **ND‑paced** — slow, predictable, gentle  
- **Non‑coercive** — participation is optional  
- **Non‑interpretive** — no meaning assigned to actions  

These principles protect learners and the learning environment.

---

# 🧭 **2. Instructor Compass**
Before teaching, instructors perform a Compass check.

### **North — Structure**  
Know the session plan, even if it’s simple.

### **South — Safety**  
Ensure the environment is calm, predictable, and non‑pressured.

### **East — Tone**  
Set a gentle tone for the session.

### **West — Cosmology**  
Keep cosmology outside the classroom unless explicitly teaching it.

### **Center — Pacing**  
Check your own pacing and the group’s pacing.

Use:  
- **Compass North**  
- **Compass South**  
- **Compass East**  
- **Compass West**  
- **Compass Center**

# 🗺️ **3. Teaching Maps**
Maps are the foundation of UMM instruction.

### Instructor Guidelines
- Introduce one map at a time  
- Keep maps symbolic  
- Use nodes, drift lines, regions, boundaries  
- Avoid complex geography  

### Classroom Map Flow
1. Place one node  
2. Add one drift line  
3. Mark one region  
4. Add one reversible boundary

# 📘 **10. Instructor Troubleshooting**
### **If the group feels overwhelmed:**  
- shrink the field  
- reduce to one symbol  
- pause  
- reset  

### **If the group drifts:**  
- return to the Compass  
- redraw boundaries  
- simplify the map  

### **If pacing collapses:**  
- use Zen Reset  
- remove symbols  
- slow transitions  

### **If someone needs space:**  
- allow quiet participation  
- offer reversible actions  
- avoid spotlighting
