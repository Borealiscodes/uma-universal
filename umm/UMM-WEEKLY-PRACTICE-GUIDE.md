# 🌱 **Weekly Structure Overview**
Each day focuses on **one gentle domain** of UMM:

- **Monday — Stillness & Compass**  
- **Tuesday — Maps**  
- **Wednesday — Worlds**  
- **Thursday — Characters**  
- **Friday — Play Tree**  
- **Saturday — Multiplane Interaction**  
- **Sunday — Integration & Rest**

Every day uses the same soft pacing:

- one breath  
- one anchor  
- one symbol  
- one motion  
- one check  
- one reset  

This keeps the week calm and reversible.

# 🌤️ **MONDAY — Stillness & Compass**
### *Goal: Set tone and pacing for the week.*

**Practice:**

1. **Stillness** — one breath, one anchor dot  
2. **Compass North** — “Where am I?”  
3. **Compass South** — “Am I safe?”  
4. **Compass East** — choose tone  
5. **Compass West** — keep cosmology outside  
6. **Compass Center** — check pacing  

Use:  
- **Return to Stillness**  
- **Compass Center**
