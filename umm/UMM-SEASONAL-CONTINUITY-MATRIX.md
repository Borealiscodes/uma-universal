# 🧭 **2. Seasonal Continuity Overview**
Each season contributes one continuity element and releases complexity.

### **Spring → Summer**
Carry: one node, one drift line  
Release: unused symbols

### **Summer → Autumn**
Carry: one region, one shimmer  
Release: second characters

### **Autumn → Winter**
Carry: one simplified boundary, one pattern  
Release: large regions

### **Winter → Spring**
Carry: one anchor, one tone  
Release: everything else

This is the **seasonal continuity arc**.
