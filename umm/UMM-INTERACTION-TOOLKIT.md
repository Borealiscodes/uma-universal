# 🌱 **1. Soft Interaction Principles**
Before interacting, UMM uses five gentle principles:

- **Reversible** — every action can be undone  
- **Symbolic** — nothing is literal or emotional  
- **ND‑paced** — slow, calm, no overload  
- **Non‑coercive** — nothing forces you  
- **Non‑interpretive** — nothing assumes meaning  

These principles keep interaction safe.
