# 🔒 **2. Boundary Properties**
All boundaries in UMM share five constitutional properties:

- **Reversible** — boundaries can be removed without consequence  
- **Symbolic** — boundaries do not imply meaning  
- **ND‑paced** — boundaries are introduced slowly  
- **Non‑coercive** — boundaries never restrict the learner  
- **Non‑interpretive** — boundaries do not encode narrative  

Boundaries protect structure, not emotion.
